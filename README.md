# QA Automation Training

This repository contains my practice work during the QA Automation Testing training.
The training focuses on **Automation Testing**, where I will learn how to design and execute automated test cases using modern testing tools. The main automation tool used in this training is **Cypress**.

## Automation Tool

The automation tests in this training will be implemented using:
- Cypress : for end-to-end web automation testing
- Git & GitHub : for version control and collaboration

## Test Application

The website used for automation practice is:
Booking Website: https://www.booking.com
The automation tests will focus on validating important functionalities such as:
- Searching for stays
- Applying filters
- Viewing property details
- Verifying search results

## Training Goals

During this training I aim to:
- Learn automation testing using Cypress
- Write automated test scripts
- Understand Git workflow (Repository, Commit, Branch, Pull Request)
- Practice testing real-world web applications
